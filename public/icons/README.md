Drop your app icon artwork here as these exact filenames:

- `icon-192.png` — 192×192px, used for Android home screen
- `icon-512.png` — 512×512px, used for Android splash/maskable icon
- `apple-touch-icon.png` — 180×180px, used for iOS home screen

Use a square image with no transparency (iOS ignores transparency and shows black instead). Once added, commit and push — Vercel will redeploy automatically and the home screen icon will update.
